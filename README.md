# IoT Home Automation System using ESP32

## 📌 Description
This project implements a Wi-Fi-based home automation system using ESP32. It allows users to control electrical appliances such as lights and fans remotely via a web interface.

## 🚀 Features
- Wireless control using Wi-Fi
- Low-cost hardware setup
- Real-time response (low latency)
- Simple web-based UI
- Safe AC switching using relay module

## 🛠️ Tech Stack
- ESP32 (Microcontroller)
- Arduino IDE
- Embedded C/C++
- Wi-Fi Networking

## 🔌 Hardware Required
- ESP32
- Relay Module (2/4 channel)
- Jumper wires
- Breadboard
- AC appliances (bulb/fan)

## ⚙️ Setup Instructions
1. Install Arduino IDE
2. Add ESP32 board support
3. Upload the code to ESP32
4. Replace Wi-Fi SSID & Password
5. Open Serial Monitor → Get IP address
6. Enter IP in browser

## 🌐 Usage
- Connect to same Wi-Fi network
- Open ESP32 IP in browser
- Toggle appliances ON/OFF

## ⚠️ Safety Note
Ensure proper isolation while handling AC loads. Use relay modules with optocouplers for safety.
